# 🧱 miconsole-maven

Proyecto de consola en **Java con Maven**, orientado a práctica de estructura, compilación y ejecución de aplicaciones simples.

![Java](https://img.shields.io/badge/Java-blue?logo=openjdk) ![Maven](https://img.shields.io/badge/Maven-3.x-orange?logo=apachemaven)

---

## 📌 Objetivo

Ofrecer una base simple para practicar desarrollo de consola sobre una estructura estándar Maven.

## ▶️ Ejecutar

```bash
git clone git@github.com:Carl0gonzalez/miconsole-maven.git
cd miconsole-maven
mvn clean compile
```

## 💡 Valor del repositorio

Sirve como ejercicio base para reforzar flujo Java, estructura de carpetas y ciclo de build.
